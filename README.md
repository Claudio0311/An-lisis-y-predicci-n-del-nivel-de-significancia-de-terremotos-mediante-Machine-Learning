# 🧠Predicción del nivel de significancia de terremotos

## 🎯Objetivo:
Desarrollar e implementar un modelo de Machine Learning y Deep Learning enfocado en el procesamiento de variables sísmicas para predecir la variable de significancia de terremotos.
## 🗂️DataSet:
El modelo fue entrenado y evaluado utilizando el Earthquake Dataset de Kaggle, un conjunto de datos globales diseñado para el estudio y consolidación de magnitudes, alertas ambientales y variables de precisión geofísica.
